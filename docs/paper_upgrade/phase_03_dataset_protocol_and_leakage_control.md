## Changes Made

1. Updated `src/costtrace/modeling/graphsage.py`.
   - Replaced node-level stratified splitting with household-level group splitting.
   - Removed `is_index` and `sus_enc` from GraphSAGE features because they deterministically proxy SARS positivity in the current artifacts.
   - Switched GraphSAGE continuous features from pre-normalized full-data columns to raw centrality/contact-duration columns scaled from train households only.
   - Added split protocol, removed features, train-only scaling, split counts, and household split metadata to model outputs.

2. Added `scripts/18_phase03_leakage_audit.py`.
   - Generates dataset counts, timestamp availability counts, legacy node-split overlap counts, proxy-feature summaries, current household split overlap checks, and a data dictionary mapping artifact.
   - Avoids exporting participant-level rows or identifiers in audit tables.

3. Updated `reports/costtrace_research_manuscript.tex`.
   - Expanded dataset and preprocessing sections using repository evidence.
   - Added a reproducible preprocessing pipeline.
   - Added a leakage-control and split-protocol subsection.
   - Added a leakage audit table.
   - Updated GraphSAGE metrics and intervention results after rerunning the leakage-controlled model.
   - Added evidence-based Ethics, Data Availability, and AI/Tool Use wording with explicit TODOs for missing official inputs.

4. Added `docs/paper_upgrade/phase_03_leakage_audit_table.md`.
   - Markdown leakage audit table generated by the phase audit script.

5. Regenerated model, intervention, audit, and chart artifacts.

## Remaining Risks

- GraphSAGE performance dropped after leakage control, so earlier claims of strong GNN superiority are no longer supported.
- The current household split is a single split; metric variance is unknown until Phase 04 multi-fold evaluation.
- Counterfactual evaluation still uses observed transmission labels and must remain framed as retrospective observed-edge blocking.
- Data availability and ethics text are not submission-final until official source and access terms are confirmed.
